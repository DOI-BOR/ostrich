1 Running OSTRICH

OSTRICH can be utilized across computing environments on numerous operating systems. For most common operating systems, builds of OSTRICH already exist that can be utilized with minimal effort. See "Running an Analysis" for more information on how to deploy OSTRICH on various computing platforms.

2 Building OSTRICH

For uncommon operating systems or when the OSTRICH source code as been customized, the toolkit will need to be recompiled prior to use. See "Building OSTRICH" for more information on how to build OSTRICH on various computing platforms.

Building OSTRICH is an advanced task that is not recommended for those who are unfamiliar with software development. While utilities are provided that simplify the build process, it it strongly recommended that precompiled versions of OSTRICH are first utilized.
