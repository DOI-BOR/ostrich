Examples

Examples using OSTRICH for calibration and optimization

Initial Publication:

April 11, 2021

Modified:

April 11, 2021

The examples below demonstrate various, but not all, aspects of OSTRICH. Required input files and model executables are provided with the OSTRICH download within the "examples" directory. For some of the examples, users may need to edit batch files (e.g. mpirun.bat or OSTRICH.bat) to adjust the path for OSTRICH.exe to reflect the location of their OSTRICH installation.
