Algorithms - Sampling Algorithm (Big Bang - Big Cruch)

The OSTRICH sampling optimization algorithm

Initial Publication:

September 16, 2020

Modified:

September 16, 2020

The following optional group will configure the Big Bang – Big Crunch (BB-BC) algorithm and will be processed if [ProgramType] is set to “SamplingAlgorithm”. 

BeginSamplingAlg
MaxEvaluations  
EndSamplingAlg

BeginSamplingAlg
MaxEvaluations      100
EndSamplingAlg

Figure 1: General Format (left) and Example (right) of the BB-BC Group

Where BeginSamplingAlg and EndSamplingAlg are parsing tags that wrap the following set of algorithm configuration variables:

  • MaxEvaluations: The computational budget. The default value is 100.

References

Erol, O. K.,Eksin, I. 2006. A new optimization method: Big Bang-Big Crunch. Advances in Engineering Software 37, 106-111.
